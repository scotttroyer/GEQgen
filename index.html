GEQgen

A web-based XML code generator for PreSonus StudioLive GEQ presets (v1.1)

What is this?

This web app makes GEQ presets for first generation PreSonus StudioLive mixers. For more information, read this article.

31-Band Graph (values in dB)

1. Create the EQ Curve

The decibel value for each frequency band can be typed in directly or incremented up and down with the arrow keys ↑ and ↓. Tab ↹ and Shift+Tab move right and left through the frequency bands. Values can range from a maximum 15.0 dB to a minimum -15.0 dB in increments of 0.1 dB. Invalid values will highlight in red.

Preset Name

2. Assign a Preset Name

Give your GEQ preset a name. Only letters, numbers, and spaces are allowed (for now). Maximum limit 30 characters.
