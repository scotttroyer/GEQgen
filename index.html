GEQgen

A web-based XML code generator for PreSonus GEQ presets
Navigation

The decibel value for each frequency band can be typed in directly or incremented up and down with the arrow keys ↑ and ↓. Tab ↹ and Shift+Tab move right and left through the frequency bands. Values can range from a maximum 15.0 dB to a minimum -15.0 dB in increments of 0.1 dB. Invalid values will highlight in red.

XML

Click on the XML code that appears to the left. Copy and paste that into a text file. Save it with same preset name plus ".xml" (e.g. "My First Preset.xml").
